Security Grants and Audit Bounty Program

This page is a public governance proposal source for the Steward council proof. It tests whether an autonomous delegate can approve a concrete public-goods security program while keeping the decision tied to explicit funding and recipient constraints.

Proposal summary

Approve a 750,000 USDC security grants and audit bounty program for community builders, independent auditors, open-source maintainers, and whitehat researchers. The program funds vulnerability research, audit contests, security tooling, and remediation grants.

Decision facts

Requested action
Approve security grants and audit bounty program
Requested budget
750,000 USDC maximum
Funding window
Six months
Recipient type
Community builders, independent auditors, maintainers, and whitehat researchers
Risk note
No team token unlock, no insider allocation, and no irreversible protocol change

Rationale

  • The budget is below 1,000,000 USDC.
  • The spend is a community/public-goods grant program.
  • Funds are milestone-based and require public post-mortems for critical payouts.
  • No foundation team token vesting or insider liquidity is changed.
